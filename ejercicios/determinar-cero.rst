Determinar 0
------------

Determinar si el número ingresado es *0*

 * **Entrada**

   Un número cualquiera

 * **Salida**

   Si el número es 0.

 * Comportamiento::

    Ingrese un número: 2

    Ingrese un número: 0
    Su número es cero

 * Solución::

    num = input("Ingrese un número")
    if num == 0:
        print ("Su número es cero")
