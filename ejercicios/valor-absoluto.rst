Valor Absoluto
---------------

Realizar un programa que pueda determinar el
valor absoluto de un número ingresado.

Por definición el valor absoluto cumple lo siguiente:

.. math::

    |x| = \left\{ \begin{array}{rl}
            -x &\mbox{ if $x<0$} \\
             x &\mbox{ en otro caso}
          \end{array} \right.


* **Entrada**

    Un número real *n*.

* **Salida**

    El valor absoluto de *n*.

* Comportamiento::

    Ingrese número: 67
    67

    Ingrese número: -24
    24

    Ingrese número: 0
    0
