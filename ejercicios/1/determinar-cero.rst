Determinar 0
------------

Determinar si el número ingresado es *0*

Al programa se le debe ingresar
un número *n* cualquiera, y debe
entregar si es cero o no.

* Comportamiento

::

    Ingrese un número: 2

::

    Ingrese un número: 0
    Su número es cero

* Solución

::

    num = input("Ingrese un número")
    if num == 0:
        print ("Su número es cero")
