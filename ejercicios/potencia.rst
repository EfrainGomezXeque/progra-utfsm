Potencia
---------

Realizar un programa que pueda determinar la
potencia de un número ingresado por el usuario *a*,
igresando también, el exponente *b*.

.. math::

    a^{b}


* Comportamiento

::

    Ingrese base: 2
    Ingrese exponente: 3
    8

::

    Ingrese base: 3
    Ingrese exponente: 2
    9
