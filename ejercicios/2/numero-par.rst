Número par
----------

Desarrolle un programa
que contenga una función
llamada:

::

	par(x)

que retorne **True** si *x* es par,
y **False** si es impar.

Por ejemplo:

::

	par(16)
	True
	par(29)
	False
	par('hola')
	Traceback (most recent call last):
	TypeError: not all arguments converted during string formatting
