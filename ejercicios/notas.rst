Notas
-----

#. Realizar un programa que pueda determinar si
   una nota ingresada es roja o azul.

 * **Entrada**

    una nota de una prueba (un valor *int* entre 0 y 100).

 * **Salida**

    *rojo* o *azul*, dependiendo si la nota es mayor o menor a 55.

 * Comportamiento::

    Ingrese nota: 45
    rojo
    Ingrese nota: 60
    azul
