Introducción a C
================

Este apunte está diseñado
para estudiantes que ya han pasado
por la asignatura de programación en Python,
y que necesitan aplicar sus conocimientos
para familiarizarse con el lenguaje C.

El apunte está organizado como un tutorial.
En cada capítulo analizaremos un programa diferente,
a través del que iremos presentando
las características del lenguaje C
de manera progresiva.
